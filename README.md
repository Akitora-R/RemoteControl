# 非侵入式远程控制方案 / Non-Intrusive Remote Control Solution
## 项目概述 / Project Overview

本项目基于WinForm开发，利用视频采集卡和CH9329芯片实现了一种非侵入式的远程控制方案。通过视频采集卡获取远程设备的屏幕图像，并结合CH9329芯片模拟键盘和鼠标操作，用户可以在不直接接触远程设备的情况下进行远程控制。

This project is developed based on WinForm, utilizing a video capture card and the CH9329 chip to implement a non-intrusive remote control solution. By capturing the screen image of the remote device through the video capture card and simulating keyboard and mouse operations with the CH9329 chip, users can remotely control the device without direct physical interaction.

## 功能特性 / Features

- 视频采集：通过视频采集卡实时获取远程设备的屏幕图像。

- 键鼠模拟：使用CH9329芯片模拟键盘和鼠标操作，实现对远程设备的控制。

- 非侵入式：无需在远程设备上安装任何软件，避免对系统造成影响。

- 用户界面：基于WinForm开发，提供友好的用户界面，便于操作和监控。

- Video Capture: Real-time screen image acquisition of the remote device through a video capture card.

- Keyboard and Mouse Simulation: Simulate keyboard and mouse operations using the CH9329 chip to control the remote device.

- Non-Intrusive: No need to install any software on the remote device, avoiding any impact on the system.

- User Interface: Developed based on WinForm, providing a user-friendly interface for easy operation and monitoring.

- Operating System: Windows 10 or later.
